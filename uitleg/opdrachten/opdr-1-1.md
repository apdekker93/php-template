# Opdr. 1.1: Overal hetzelfde menu

*Onderdeel van: 1: HTML genereren met PHP*

---

Volg het onderstaande stappenplan om het menu van je website in een apart bestand te zetten. Dat bestand kan je dan op alle pagina’s invoegen op de plek waar je het menu wilt hebben!

1. Maak een nieuw bestand aan in hoofdmap, en noem het “menu.php”.
2. Uitleg: een php-bestand zal door je browser als gewoon html-bestand behandeld worden. Het enige verschil is dat een php-bestand eerst door de server verwerkt moet worden.
3. Kopieer de code voor je menu uit index.html naar menu.php.
4. Verander de naam van index.html naar index.php.
5. Verwijder het menu uit index.php, en zet daar de volgende regel neer:
   
   ```<?php include("menu.php"); ?>```

7. Typ `start-server` in de terminal. De browser opent de website automatisch.
8. Los eventuele fouten op. Lees goed wat er staat! Meestal heb je maar een klein typefoutje hebt gemaakt. Kijk dus goed na of je alle tekens goed hebt overgenomen. Vaak moet je de fout zoeken op de regel vóór het regelnummer dat je in de foutmelding leest. Veel voorkomende problemen zijn:  
    - Een spatie op een plek waar die niet hoort
    - Een “krom” aanhalingsteken in plaats van "rechte" aanhalingstekens (let op het verschil in deze zin).
    - Een puntkomma vergeten.
