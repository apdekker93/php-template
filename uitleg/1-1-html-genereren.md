# 1.1: HTML genereren

*Onderdeel van: 1: HTML genereren met PHP*

---

PHP is een taal die gemaakt is om een website te **genereren**. Dat
betekent dat je een programma kan schrijven dat een HTML-bestand voor je
schrijft. **PHP** staat voor namelijk PHP Hypertext Preprocessor. De term **Hypertext**ken je al van de HyperText Markup Language: HTML. PHP is daar dus een
"**preprocessor**" voor: een voor-verwerker. Als je een pagina nog wilt aanpassen vlak
voordat je hem naar je bezoeker op wilt sturen, biedt PHP
daarvoor een oplossing. Met PHP kan je dan bijvoorbeeld invullen hoe de
ingelogde gebruiker heet.

PHP bedenkt dus niet zelf welke HTML-tags er opgestuurd
moeten worden. Alle stukjes HTML-code moet je zelf in het PHP-programma zetten.
Door dat te doen, maak je het jezelf wel veel makkelijker om een website te
maken. Het scheelt bijvoorbeeld al veel als je het menu niet op alle pagina’s
hoeft te bewerken als er iets in veranderen moet. Zonder PHP moet je zo’n
aanpassing op iedere pagina handmatig doen. Met PHP kan je een apart bestandje
maken met de HTML-code voor je menu, en precies die code steeds in je pagina’s
plakken. Hoe doe je dat?
